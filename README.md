###GhostGrid v0.1.3.8
<br /><br />

Lightweight grid component with auto snapping & additional magic. Just add
'GhostGrid.cs' to any GameObject to activate the grid for him and his
children.

Check out 'Tools/GhostGrid' in the menu for shortcuts!
